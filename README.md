# 🌸 Floristería - Proyecto en React, Node y Tailwind

¡Bienvenido a la documentación de mi proyecto de floristería! Este repositorio contiene una aplicación web construida con **React**, **Node.js** y **Tailwind CSS**. A continuación, encontrarás toda la información necesaria para entender este proyecto.

## 📖 Descripción del Proyecto

Este proyecto es una aplicación de floristería que permite a los usuarios explorar una variedad de flores y plantas. Además, cuenta con un módulo de administración que permite gestionar productos, incluyendo la capacidad de agregar, editar y eliminar artículos de la tienda.

## 🚀 Tecnologías Utilizadas

- **React**: Biblioteca de JavaScript para construir interfaces de usuario.
- **Node.js**: Entorno de ejecución para JavaScript en el servidor.
- **Tailwind CSS**: Framework de CSS para un diseño moderno y responsivo.
- **JSON**: Utilizado para almacenar y gestionar los datos de los productos.

## 🧑‍💻 Módulo de Administración

El módulo de administración permite a los usuarios gestionar los productos de la tienda. Las funcionalidades incluyen:

- **Agregar productos**: Permite añadir nuevos productos a la tienda.
- **Editar productos**: Modifica la información de los productos existentes.
- **Eliminar productos**: Elimina productos de la tienda.

## 📊 Datos

Los datos de los productos se almacenan en un archivo JSON ubicado en la carpeta Back-end `/data/catalogo_productos.json`. Este archivo contiene información sobre cada producto, como nombre, precio, descripción y URL de la imagen.
